## Set up config

We add 
1. Router
2. CSS-preprocessor(node-sass)
3. ESLint+Prettier
4. All configs in package.json

In `package.json` add this: 

```
"postcss": {
    "plugins": {
      "autoprefixer": {}
    }
  },
```
